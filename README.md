# Data-analyst-Intern-Task-2
Tools Used:

Power BI – Dashboard creation and visualization
Microsoft Excel / CSV – Data source and preprocessing
Dataset: Superstore Sales Dataset

The dataset contains information about:
Orders
Sales
Profit
Customers
Products
Categories and Sub-Categories
Regions and States
Shipping Details
The dashboard includes visualizations such as:
Total Sales Analysis
Profit Analysis
Sales by Region
Sales by Category and Sub-Categors

Some of the business insights identified from the dashboard include:

Identification of the highest revenue-generating regions.
Analysis of profitable and loss-making product categories.
Understanding sales trends over time.
Recognition of top-performing products and customer segments.
Detection of areas requiring business improvement.
The dashboard was designed following these storytelling principles:

1. Choosing the right chart for each analysis.
3. Avoiding unnecessary visual clutter.
4. Highlighting key findings using visual emphasis.
5. Providing business context for each visualization.
6. Focusing on actionable insights rather than only displaying data.

 I Learning from this Task:

Through this project, I learned:
Data visualization best practices.
Dashboard design principles.
Interactive reporting in Power BI.
Business insight generation from sales data.
Effective data storytelling techniques.
